ESG Dashboard (Excel)

An interactive ESG (Environmental, Social, and Governance) dashboard built in Microsoft Excel to monitor sustainability performance across companies and reporting periods. The dashboard converts raw ESG data into meaningful insights using KPIs, automated calculations, and dynamic visualizations.

✨ Features

Interactive slicers for company and date filtering

Automated KPI calculations and ESG scoring

Executive-style charts and performance summaries

Comparative company performance analysis

Structured dataset for easy updates and scalability

Integrated ESG category scoring (Environmental, Social, Governance)

 🛠️ How to Use

Download the Excel file from the repository.

Open the file in Microsoft Excel (Excel 2016 or newer recommended).

Navigate to the Dashboard worksheet.

Use the slicers to filter by:

Company

Date / Reporting Period

KPIs and charts update automatically based on your selection.

📂 File Structure

ESG Dashboard.xlsx → Main dashboard and dataset

Source_Data → Raw ESG operational data

Calculation → KPI calculations and scoring logic

Dashboard → Visualizations and summary insights

ESG Log → Supporting ESG tracking records
